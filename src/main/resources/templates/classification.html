Classification Models

K-Nearest Neighbors (KNN)

KNN classifies new data by finding its closest neighbors in the training data, assigning the most frequent class among those neighbors.

Decision Tree

A decision tree classifier splits data into increasingly specific segments based on features, ultimately assigning a class label at the leaf node of the tree.

Support Vector Classifier (SVC)

SVC carves a hyperplane in high dimensions to optimally separate data points into distinct classes.

Logistic Regression

Logistic regression predicts the probability of an event happening (like yes/no or 0/1) by analyzing data and fitting an S-shaped curve.
