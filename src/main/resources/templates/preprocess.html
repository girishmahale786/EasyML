Data Preprocessing

Impute Missing Data

When your data has missing values, imputation fills them in. You can use various strategies to predict the missing value based on other data points.

Delete Missing Data

If you have missing data points, you can just remove them entirely from your dataset. This is appropriate if the missing data is minimal and won't significantly impact your analysis.

Data Encoding

Categorical data can't be directly used by ML models. Encoding converts these categories into numerical values the models can understand.

Feature Scaling

ML algorithms can be sensitive to the scale of your data. Feature scaling transforms your features to a common range. This ensures all features contribute equally to the model.
