Data Visualization

Line Plot

A line plot is a graph that uses dots or symbols on a number line to show how often each data value appears. It's a simple way to see how your data is spread out.

Bar Plot

A bar plot uses rectangular bars to display comparisons between categories. The length or height of each bar represents a specific value.

Scatter Plot

A scatter plot uses dots to visualize the relationship between two numerical variables, with each dot representing a single data point.

Box Plot

A box plot uses a box to show the middle half of your data and whiskers to show the spread. It also highlights potential outliers using quantiles.

Histogram

A histogram uses bars to show how many data points fall within certain ranges, giving a quick view of your data's distribution.

Heatmap

A heatmap uses color intensity to represent data values in a grid, making it easy to see areas of high and low values at a glance.
