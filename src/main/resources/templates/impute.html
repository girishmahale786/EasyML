Data Imputation

Mean Imputation

It fills missing data points with the average value of the existing data in that column. It's simple and maintains sample size, but underestimates variability and can bias analysis towards the mean.

Mode Imputation

It replaces missing values with the most frequent value in the column. It's fast and works well for categorical data, but can mask underlying trends and affect analysis if the data isn't non-uniform.

Median Imputation

It fills missing data with the middle value of a sorted list within a column. It's robust to outliers unlike mean imputation, but assumes the data is symmetrical and can underestimate variability.

Backward Fill

It replaces missing values with the next available value in the same column. It's efficient and keeps data size intact, but can be misleading if the data isn't truly sequential and may not capture underlying trends.

Forward Fill

It replaces missing values with the previous available value in the same column. It's simple and maintains data size, but assumes the missing value reflects the previous state, which can potentially introduce bias.
