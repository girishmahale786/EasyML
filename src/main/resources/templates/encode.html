Data Encoding

One-Hot Encoding

One-hot encoding transforms categorical data into binary vectors. Each category gets its own column with a 1 for presence and 0s elsewhere. This avoids order assumptions but increases data dimensionality.

Label Encoding

Label encoding converts categorical data labels (e.g., colors) into numerical values (e.g., red = 0, green = 1). This simplifies data for algorithms but can introduce misleading order (red isn't inherently better than green).
