Regression Models

Linear Regression

By analyzing data points, linear regression creates a straight line that best predicts how one variable changes along with another.

Polynomial Regression

Polynomial regression goes beyond straight lines, fitting a curved model to capture non-linear relationships between variables.

Lasso Regression

Lasso regression shrinks coefficients in linear regression, promoting simpler models and potentially selecting key features.

Ridge Regression

Ridge regression tackles overfitting in linear regression by adding a penalty that shrinks the coefficients of the model, leading to a more stable model.
